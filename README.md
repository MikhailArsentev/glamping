# Сайт глэмпинга "Тропос"

Современный многостраничный сайт глэмпинга, созданный на Nuxt 3 с использованием Tailwind CSS и PrimeVue.

## Описание проекта

Сайт глэмпинга "Тропос" включает в себя:
- **Главная страница** - Hero секция, карточки преимуществ, информация о глэмпинге
- **О глэмпинге** - История создания с таймлайном, миссия, фотогалерея
- **Проживание** - Каталог домиков с карточками и формами бронирования
- **Питание** - Информация о питании и форма заявки
- **Досуг** - Доступные активности и развлечения
- **Контакты** - Контактная информация и форма обратной связи
- **Политика конфиденциальности** - Условия обработки персональных данных

## Технологический стек

- **Framework**: Nuxt 3 (v4.2.2)
- **Styling**: Tailwind CSS 3.x
- **UI Library**: PrimeVue 3.x
- **Animations**: @vueuse/motion
- **Forms**: PrimeVue встроенные компоненты
- **Icons**: PrimeIcons

## Основные возможности

### Анимации
- Scroll-triggered анимации при прокрутке страницы
- Плавные переходы между элементами
- Анимированные счетчики статистики

### Формы
- Форма бронирования с выбором дат, количества гостей и контактными данными
- Упрощенная форма заявки (имя и телефон)
- Валидация полей
- Toast уведомления об успехе/ошибке

### Адаптивность
- Полностью responsive дизайн
- Мобильное меню с боковой панелью
- Оптимизация под различные устройства

### UI Компоненты
- Карточки домиков с галереей изображений
- Таймлайн истории создания глэмпинга
- Фотогалереи с grid layout
- Кнопки призыва к действию

## Деплой на продакшн

### Автоматический деплой через GitHub Actions

Проект настроен для автоматического деплоя через GitHub Actions.

**Подробная документация:** [DEPLOYMENT.md](DEPLOYMENT.md)

#### Быстрый старт:

1. **Настройте сервер:**
   ```bash
   wget https://raw.githubusercontent.com/your-repo/glamping/main/scripts/setup-server.sh
   chmod +x setup-server.sh
   ./setup-server.sh
   ```

2. **Добавьте GitHub Secrets:**
   - `SSH_PRIVATE_KEY` - приватный SSH ключ
   - `SERVER_HOST` - IP адрес сервера
   - `SERVER_USER` - имя пользователя
   - `APP_PATH` - путь к приложению на сервере

3. **Сделайте push в ветку `main`** - деплой запустится автоматически!

### Ручной деплой

```bash
export SERVER_USER="ubuntu"
export SERVER_HOST="your-server-ip"
export APP_PATH="/var/www/glamping"

chmod +x scripts/deploy-manual.sh
./scripts/deploy-manual.sh
```
